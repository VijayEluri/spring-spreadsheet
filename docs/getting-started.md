# Getting Started with Spring Spreadsheet

Include dependency in your maven project:

`pom.xml`
{% highlight text %}
    <dependency>
        <groupId>org.springframework.spreadsheet</groupId>
        <artifactId>spring-spreadsheet</artifactId>
        <version>0.0.1-SNAPSHOT</version>
    </dependency>
{% endhighlight %}
